This is a simple mathematical processor that was made as an example of using OOP in tasks like string parsing.
This math processor can calculate expressions like "2+3*4".

* Usage: calc "29.78 + 50 * 88.25^5 - sqr( sqrt( 27 ) + 5 / 25 )"

* Version 0.1 supports folowing actions:

"-" - Subtraction
"+" - Adding
"*" - Multiplication
"/" - Division
"^" - Exponentiation
"sqr()" - Square
"sqrt()" - Square root

Compilation
$ g++ calc.cpp -o calc
